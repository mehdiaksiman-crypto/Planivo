## 🇫🇷 FRANÇAIS

### Qu'est-ce que Planivo ?
Planivo est un planificateur personnel numérique complet, installable sur votre téléphone, tablette ou ordinateur — sans App Store, sans abonnement, sans compte requis.

### ✅ Fonctionnalités incluses
- 📅 Vue annuelle, mensuelle et journalière
- ✅ Gestion des tâches avec dates d'échéance
- 💰 Suivi des dépenses par catégorie
- 💵 Suivi des revenus par catégorie
- 📊 Budget mensuel avec conseils financiers
- 👤 Profil personnel, conjoint et enfants avec calcul d'âge automatique
- 🎨 Personnalisation complète (couleurs, nom, devise, fond)
- 💾 Export/Import de vos données (fichier JSON)
- 📱 Transfert entre appareils via Export/Import JSON
- 🌐 Bilingue Français / Anglais

---

### 📱 INSTALLATION SUR TÉLÉPHONE

**iPhone / iPad (Safari uniquement) :**
1. Ouvrez le lien Planivo dans **Safari**
2. Appuyez sur l'icône **Partager** (carré avec flèche vers le haut)
3. Faites défiler et appuyez sur **"Sur l'écran d'accueil"**
4. Appuyez sur **"Ajouter"**
5. Planivo apparaît comme une vraie app sur votre écran d'accueil ✅

**Android (Chrome) :**
1. Ouvrez le lien Planivo dans **Chrome**
2. Un bouton **"📲 Installer l'app"** apparaît automatiquement en bas
3. Appuyez dessus et confirmez
4. Planivo s'installe sur votre écran d'accueil ✅

> ⚠️ Sur iPhone, utilisez Safari et non Chrome pour l'installation.

---

### 💻 UTILISATION SUR ORDINATEUR

1. Ouvrez le lien Planivo dans votre navigateur (Chrome recommandé)
2. Utilisez directement dans le navigateur, ou
3. Cliquez sur l'icône d'installation dans la barre d'adresse pour l'installer comme app de bureau

---

### 💾 SAUVEGARDER VOS DONNÉES

Vos données sont stockées localement sur votre appareil.
**Important :** Si vous videz le cache ou les cookies de votre navigateur, vos données seront effacées.

**Pour éviter toute perte :**
1. Allez dans **Paramètres** → **💾 Sauvegarde & Restauration**
2. Cliquez **"📥 Exporter mes données"**
3. Un fichier `agenda-sauvegarde-DATE.json` sera téléchargé
4. Conservez ce fichier précieusement

**Pour restaurer :**
1. Allez dans **Paramètres** → **💾 Sauvegarde & Restauration**
2. Cliquez **"📤 Restaurer mes données"**
3. Sélectionnez votre fichier de sauvegarde

---

### 📱 TRANSFÉRER ENTRE APPAREILS

1. Sur l'appareil source : **Paramètres** → **"📥 Exporter mes données"** → un fichier `.json` se télécharge
2. Transférez ce fichier sur l'autre appareil (WhatsApp, email, câble USB...)
3. Sur l'appareil destination : **Paramètres** → **"📤 Restaurer mes données"**
4. Sélectionnez le fichier `.json` → vos données sont transférées ✅

### 💾 SAVING YOUR DATA

Your data is stored locally on your device.
**Important:** If you clear your browser cache or cookies, your data will be erased.

**To prevent data loss:**
1. Go to **Settings** → **💾 Backup & Restore**
2. Click **"📥 Export my data"**
3. A file `agenda-backup-DATE.json` will be downloaded
4. Keep this file safe

**To restore:**
1. Go to **Settings** → **💾 Backup & Restore**
2. Click **"📤 Restore my data"**
3. Select your backup file

---

### 📱 TRANSFER BETWEEN DEVICES

1. On the source device: **Settings** → **"📥 Export my data"** → a `.json` file downloads
2. Transfer this file to the other device (WhatsApp, email, USB cable...)
3. On the destination device: **Settings** → **"📤 Restore my data"**
4. Select the `.json` file → your data is transferred ✅

Planivo collects no data. Everything stays on your device.
No account required. No internet connection needed after installation.
